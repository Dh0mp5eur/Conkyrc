## Getting Started Guide

## How to setup:

		
#### Git Clone

If you have Git installed you can clone the repo

	git clone https://github.com/Dh0mp5eur/Conkyrc.git

#### Copy

	cp .conkyrc ~/
	cp -r .conky ~/

#### Set Position

Change gap_x and gap_y in .conkyrc for set position.
